## How to run
* `python -m venv venv`
* `source venv/bin/activate`
* `pip install -r requirements.txt`
* `cd project`
* change `./project/settings.py`
* `python manage.py migrate`
* `python manage.py runserver`
* `cd fontend`
* `npm start`
